# Personal Finance Tracker

## Description
A beginner-friendly finance tracker to manage income and expenses, calculate monthly savings, and view a simple dashboard. Built using **HTML, CSS, and Java**, this project simulates backend logic with Java objects.

## Features
- Add income and expense entries via Java console
- View all transactions with type, description, and amount
- Calculate total income, total expenses, and savings
- Static HTML/CSS dashboard to visualize financial summary

## Tech Stack
- **Frontend:** HTML, CSS
- **Backend:** Core Java
- **Data Storage:** Java ArrayList (In-memory)
